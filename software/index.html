Project Statuses

All projects have either the following development statuses:

  • Current
    • Rolling Release: In active development with a predictable version cycle.
    • Active: In active development with no planned dates for releases or changes.
    • Queued: Paused development, no time set to continue.
  • Previous
    • Maintenance Only: Receives bug and security fixes only.
    • Archived: No development what so ever. A project may be rebooted in the future.
